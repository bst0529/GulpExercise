第三方模塊-Gulp

  • 使用npm install gulp下載gulp庫文件
  • 在專案根目錄下建立gulpfile.js文件
  • 重構專案的資料夾結構 src目錄放置原始程式碼檔 dist目錄放置構建後檔
  • 在gulpfile.js檔中編寫任務
    1. 安裝gulp命令行工具 => npm install gulp-cli -g
    2. 在命令列工具中執行gulp任務

Gulp中提供的方法

  • gulp.src()：獲取任務要處理的檔
  • gulp.dest()：輸出檔
  • gulp.task()：建立gulp任務
  • gulp.watch()：監控檔的變化

Gulp插件

  • gulp-htmlmin ：html檔案壓縮
  • gulp-csso ：壓縮css
  • gulp-babel ：JavaScript語法轉化
  • gulp-less: less語法轉化
  • gulp-uglify ：壓縮混淆JavaScript
  • gulp-file-include 公共檔包含
  • browsersync 流覽器即時同步
